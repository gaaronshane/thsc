    Spreadsheet Conversion Tool

    Roxy

    Spreadsheet Conversion Tool works with various types of spreadsheet formats. Application allows you easily converts csv, et, numbers, xls, xlsx, ods and ...other spreadsheet files to png, pdf, html, jpg and other formats. Friendly, clear and convenient design makes working with application easy and understandable. Full list of formats available for conversion: *From Spreadsheet file to file - csv -> html, pdf, jpg, png, ods, xls, xlsx - et -> csv, html, ods, pdf, xls, xlsx - numbers -> html, pdf, jpg, png, csv, xls, xlsx - ods -> html, pdf, jpg, png, csv, xls, xlsx - sdc -> html, pdf, jpg, png, csv, ods, xls, xlsx - xls -> html, pdf, jpg, png, xps, csv, ods, xlsx - xlsm -> html, pdf, jpg, png, xps, ods, xls, xlsx - xlsx -> html, pdf, jpg, png, xps, csv, ods, xls *From file to Spreadsheet file - csv <- et, numbers, ods, sdc, xls, xlsx - ods <- csv, et, sdc, xls, xlsx, xlsm - xls <- csv, et, numbers, ods, sdc, xlsx, xlsm - xlsx <- csv, et, numbers, ods, sdc, xls, xlsm
